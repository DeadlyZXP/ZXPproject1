关于我们 About us
我们创始于2009年，随着业务扩展，2013年我们联合伙伴共同创建了南京电子商务公司,并重新塑造了某某互动品牌,正式开始了公司化运作。一路走来,我们经历风雨,但我们始终坚持。我们一直奉行用户至上原则,无论在视觉设计，代码开发还是产品运营,用户的体验始终是我们第一追求。从产品设计,开发到运营，团队聚集了互联网各个领域的专家。除了提供核心的移动APP，WEB设计和开发服务外。我们还将品牌创意，宣传推广包括产品运营服务进行融合，为广大企业事业客户提供全方位一体化的解决方案。
